# Tmux チートシート

## 基本操作

### セッション管理
- `tmux` → 新しいセッションを開始
- `tmux new -s <name>` → 名前付きセッションを作成
- `tmux ls` → セッション一覧
- `tmux attach -t <name>` → セッションにアタッチ
- `<C-b> d` → セッションからデタッチ

### 設定
- デフォルトシェル: `zsh`
- マウス操作: 有効
- コピーモード: `vi`キーバインド
- Escキー応答: 高速化済み

## ウィンドウ操作

### 基本
- `<C-b> c` → 新しいウィンドウ作成
- `<C-b> &` → ウィンドウ削除
- `<C-b> n` → 次のウィンドウ
- `<C-b> p` → 前のウィンドウ
- `<C-b> l` → 最後のウィンドウ
- `<C-b> ,` → ウィンドウ名変更

## ペイン操作

### 分割（カスタム）
- `<C-b> s` → 水平分割（現在のディレクトリを維持）
- `<C-b> v` → 垂直分割（現在のディレクトリを維持）

### ペイン移動（Vim-like）
- `<C-b> k` → 上のペイン
- `<C-b> j` → 下のペイン
- `<C-b> h` → 左のペイン
- `<C-b> l` → 右のペイン

### ペインリサイズ（Vim-like）
- `<C-b> H` → 左に5列拡大
- `<C-b> J` → 下に5行拡大
- `<C-b> K` → 上に5行拡大
- `<C-b> L` → 右に5列拡大

### その他
- `<C-b> x` → ペイン削除
- `<C-b> z` → ペインのズーム切り替え
- `<C-b> q` → ペイン番号表示

## コピーモード

### 開始・終了
- `<C-b> [` → コピーモード開始
- `q` → コピーモード終了

### 選択・コピー（vi風）
- `v` → 選択開始
- `<C-v>` → 矩形選択
- `y` → コピー（クリップボードにも保存）
- `Enter` → コピー（クリップボードにも保存）

### 移動
- `h`, `j`, `k`, `l` → カーソル移動
- `w`, `b` → 単語移動
- `0`, `$` → 行頭・行末
- `g`, `G` → ファイル先頭・末尾

## ステータスバー

### 表示設定
- 更新間隔: 1秒
- 背景色: `#212324`
- 文字色: `#ffffff`

## マウス操作

- ペイン選択: クリック
- ペインリサイズ: ドラッグ
- スクロール: ホイール
- テキスト選択: ドラッグ

## 自動管理設定

### セッション管理
- クライアント全切断時: セッション自動削除
- サーバー自動終了: 有効
- 未接続セッション: 自動終了

## 設定ファイル
- メインファイル: `tmux/.tmux.conf`
